nimmu
====

A quick way to play with ImmutableJS from the command line.

Usage
-----

After starting the repl, you will get a typical node prompt. Use `_` to get to
Immutable.


Install
-------

**From NPM**

```bash
npm install -g nimmu
nimmu
```

**From Git**

```bash
git clone git@github.com:MrRacoon/nimmu.git
cd nimmu
npm install -g
nimmu
```
